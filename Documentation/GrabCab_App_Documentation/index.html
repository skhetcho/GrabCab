1. Introduction to GrabCab Full App

This project is a TAXI Booking Driver and Rider side full App. We have built this two Apps using React Native(Expo) for iOS and Android

Some of the main Technologies used are

  1. React Native
  2. Expo
  3. React Native Elements
  4. Firebase Realtime Database

Note: Please setup the admin portal first

2. Environment Setup

Running of a Expo App in native Platforms is very easy and simple.These are the following steps for machine setup for an Expo App.

If your Machine environment is already setup for running an Expo app then you can skip the steps from A and B.

  1. Download NodeJS
  2. Install Expo

A. Download NodeJs

    1. Download instructions
    • Mac users: For OS X, download the .pkg file form below link.(v.10.16.0 recommended)
    • Windows users: Download either the MSI or the .exe form below link, whichever you prefer .(v.10.16.0 recommended)
    Download Node v.10.16.0 from here
  2. After Installing NodeJs you'll most likely need to restart your computer.

    3. To ensure that npm is configured correctly, you can type:

    npm --version

B. Install Expo

I. In your command prompt, type:

npm install -g expo-cli
