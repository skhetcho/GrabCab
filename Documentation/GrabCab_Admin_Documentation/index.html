1. Introduction to GrabCab Admin Portal

This project is a TAXI Booking App Admin Portal. We have build this Portal using React.js for Web

Technologies Used

  1. React.Js
  2. Redux
  3. React and Redux Hooks
  4. Redux Thunk
  5. Material UI
  6. Material Table
  7. Firebase Realtime Database

    5. Admin User Creation

    Here we show just how to manually create a admin user in firebase. If you want to know how to create a firebase project please open grabcab app documentation and follow firebase setup

    Step 1 'Users' tab click on 'Add user' button and then type in your desired admin email and password and then click on 'Add user' button below as shown in picture.




    Step 2 Now mouse hover on your admin user and click on 'copy UID' and then go to 'Database' section as shown in picture.




    Step 3 Now click on the '+' button in the root and type in 'users' in key section and then again press the '+' icon of users branch and paste copied UID in key section and then again click on '+' icon of UID branch and type in 'usertype' in key section and 'Admin' in value section and then press enter or press 'Add' button as shown in picture. Also add 'isAdmin:true' in similar way.




    Now You can login in your admin portal using your email and password.
